# Rehab-rodilla
App de rehabilitación de rodilla diseñada por ClinicaFisio Sport. Incluye sesión en seco (5 bloques, 13 ejercicios) y sesión de piscina (4 bloques, 11 ejercicios), con progreso visual, verificación por ejercicio y sección de recomendaciones con señales de alerta. Pensada para usar desde el móvil.
